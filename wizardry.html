E — SVG filters

The browser's hidden image-processing lab

SVG ships a tiny node-based compositor — blur, noise, displacement, colour matrices — that you can point at any HTML element via filter:url(#id). This is where "how did they do that" usually hides.

feGaussianBlur + feColorMatrix

Gooey merge

How. Blur fuses nearby shapes, then a high-contrast alpha matrix snaps the blur back to a hard, merged blob.

<filter id="goo">
 <feGaussianBlur stdDeviation="8" result="b"/>
 <feColorMatrix in="b" values="… 0 0 0 19 -9"/>
</filter>

Liquid distortion

How. Fractal noise becomes a displacement map that pushes each pixel; animate baseFrequency and it boils.

Custom glow

How. Blur a copy of the source and feMerge it under the original — coloured bloom that box-shadow can't do on text.

Self-drawing lines

K — frameworks & libraries

The second pile: when you reach for tools

Everything above is vanilla. This pile is the wider net — reference cards, not live demos, so this page stays dependency-free. Wire any of them in when a native API hits its ceiling.

Animation & sequencing

GSAP

The industry-standard tween engine; precise timelines for DOM, SVG, canvas and WebGL.

ScrollTrigger

Scrub, pin and snap animations to scroll with cross-browser consistency.

SplitText / MorphSVG

Split text into lines/chars for stagger; morph any SVG path into another.

Motion (Framer Motion)

Spring-based animation for React/JS; runs scroll-linked work on native timelines where it can.

Anime.js

Tiny, modular engine with a built-in spring/physics system.

Theatre.js

A visual timeline editor for code — keyframe Three.js or the DOM like After Effects.

GSAP went 100% free (all former Club plugins included) in April 2025 after Webflow's acquisition — worth knowing before you reach for alternatives.
Smooth scroll & page transitions

Lenis

The de-facto smooth-scroll lib; syncs WebGL + DOM without breaking sticky or accessibility.

Locomotive Scroll

Smooth scroll + parallax; largely superseded by Lenis.

Barba.js

SPA-style animated page transitions for multi-page sites via lifecycle hooks.

Swup

Drop-in, class-driven page transitions with caching and preload.

3D · WebGL · WebGPU

Three.js

The dominant 3D library — scene graph, materials, loaders, post-processing (UnrealBloom = the glow).

React Three Fiber + drei

Three.js as declarative JSX, plus a huge helper kit (controls, HDRI, SDF text, staging).

Babylon.js

Batteries-included 3D engine: physics, PBR, GUI, node material editor, strong WebXR.

PlayCanvas

Editor-driven, collaborative engine with a tiny runtime and strong WebGPU.

curtains.js

Turns DOM images/videos into shader-animated planes that track CSS layout.

OGL

~8KB WebGL with minimal abstraction — write shaders close to the metal.

PixiJS

The fastest 2D WebGL/WebGPU renderer; great for filters and 2D effects at scale.

regl

Stateless, functional WebGL — resources and draw commands as plain data.

Creative coding & shader tooling

p5.js

Processing for the web — the friendliest on-ramp to generative art.

Paper.js

Vector scene graph on canvas with Bézier and boolean geometry.

Shadertoy

The live-coding gallery for GLSL fragment shaders — endless patterns to study (iq & co.).

The Book of Shaders

The canonical learn-GLSL resource; raymarching and SDFs for volumetric glow.

Vector & motion runtimes

Lottie

Plays After Effects animations exported as JSON — the standard for linear UI motion.

Rive

Interactive, state-machine vector animation in a tiny GPU-accelerated runtime (~10× smaller than Lottie).

Spline

Design 3D in the browser and embed it with a snippet.

tsParticles

Configurable particle/confetti/firework backgrounds (the particles.js successor).

Vanta.js

Matter.js / Rapier

2D physics for tumbling, draggable, collidable UI (Rapier is Rust/WASM and faster).

Splitting.js

Splits text/grids into CSS-variable-indexed pieces for staggered effects.
